# live

## live=true by default

```html vanilla live=true
<html>
  <div id="box">box</div>
</html>

<style>
  #box {
    border: 1px solid red;
  }
</style>

<script>
const box = document.getElementById('box');
</script>

```

## live=false

```html vanilla live=false
<html>
  <div id="box">box</div>
</html>

<style>
  #box {
    border: 1px solid red;
  }
</style>

<script>
const box = document.getElementById('box');
</script>

```
